# Dashboard 仪表盘验收标准

## 功能验收

- [x] Dashboard 布局：左侧导航 + 右侧内容区，支持折叠侧边栏
- [x] 导航侧边栏：展示菜单项，高亮当前页面，支持子菜单展开
- [x] 账户设置骨架：头像、昵称、邮箱等基础信息展示框架
- [ ] 我的技能列表：展示用户已发布的技能，支持编辑/下架/删除操作
- [ ] 发布技能：从 Dashboard 入口发布新技能，与 Entries 发布流程联动
- [ ] 数据分析图表：技能下载量、使用量、评分趋势等可视化展示
- [ ] 通知系统：系统通知、评论回复、审核结果等消息推送
- [ ] 组织管理：创建/管理团队，成员邀请，角色分配

## 安全验收

- [x] Dashboard 路由受保护（未登录跳转登录页）
- [ ] 操作权限按角色控制（creator 可发布，admin 可审核）
- [ ] 敏感操作二次确认（删除技能、转让所有权）

## UI 验收

- [x] Dashboard 整体布局响应式
- [x] 侧边栏在移动端可收起
- [ ] 数据图表自适应容器宽度
- [ ] 空状态页面设计（无技能、无通知等）
- [ ] 操作反馈（成功/失败 toast 提示）

## 性能验收

- [ ] Dashboard 首屏加载 < 2s
- [ ] 数据图表渲染 < 1s
- [ ] 技能列表分页加载流畅

## 测试验收

- [ ] Dashboard 布局组件单元测试
- [ ] 权限控制单元测试
- [ ] E2E Dashboard 导航流程测试
- [ ] E2E 技能管理流程测试
