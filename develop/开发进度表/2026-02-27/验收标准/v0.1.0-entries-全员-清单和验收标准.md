# Entries 技能注册验收标准

## 功能验收

- [x] 技能列表展示：分页加载技能卡片，展示名称、描述、FCS 评分、标签
- [x] 技能详情展示：展示技能完整信息、README、版本历史、安装说明
- [x] 搜索功能：关键词搜索技能名称和描述，结果实时更新
- [x] 标签过滤：按标签分类筛选技能列表
- [x] FCS 评分展示：技能卡片和详情页正确展示 FinClaw Score
- [ ] 技能发布：填写表单 → 上传代码/配置 → 审核 → 发布上线
- [ ] 版本管理：查看版本历史、发布新版本、版本回滚
- [ ] README 渲染：Markdown 格式 README 正确渲染（含代码高亮）
- [ ] 技能安装：一键安装/卸载技能，显示安装状态
- [ ] 评论系统：用户可对技能发表评论
- [ ] 评分系统：用户可为技能打分（1-5 星）

## UI 验收

- [x] 技能列表页响应式布局
- [x] 技能详情页响应式布局
- [x] 技能卡片设计统一
- [ ] 发布表单多步骤引导
- [ ] 版本管理 UI 清晰直观
- [ ] 搜索结果空状态提示

## 性能验收

- [x] 技能列表页首屏加载 < 2s
- [ ] 搜索响应时间 < 500ms
- [ ] 技能详情页 README 渲染 < 1s
- [ ] 列表滚动分页加载流畅

## 测试验收

- [ ] Registry 引擎单元测试覆盖 ≥80%
- [ ] 搜索功能单元测试
- [ ] E2E 技能浏览流程测试
- [ ] E2E 技能发布流程测试
