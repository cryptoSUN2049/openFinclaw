# Auth 认证系统验收标准

## 功能验收

- [x] 邮箱注册：输入邮箱+密码 → 收到验证邮件 → 点击链接激活
- [x] 邮箱登录：输入邮箱+密码 → 跳转 Dashboard
- [x] Google 登录：点击 Google 按钮 → OAuth 授权 → 回调处理 → 跳转 Dashboard
- [ ] 密码重置：点击"忘记密码" → 收到重置邮件 → 设置新密码
- [x] 登出：点击登出 → 清除会话 → 跳转首页
- [x] 会话持久化：刷新页面后保持登录状态
- [ ] Web3 钱包登录（Phase 3）

## 安全验收

- [x] JWT 有效期检查
- [x] RBAC 权限控制（5 级角色）
- [x] 受保护路由中间件
- [ ] CSRF 防护
- [ ] Rate limiting

## UI 验收

- [x] 注册页响应式布局
- [x] 错误提示清晰
- [x] 加载状态指示
