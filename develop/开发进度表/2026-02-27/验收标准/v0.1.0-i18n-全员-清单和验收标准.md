# i18n 国际化验收标准

## 功能验收

- [x] 语言切换无刷新：点击语言切换器 → 页面内容即时切换，无整页刷新
- [x] URL locale 路由：访问 /en/... 展示英文，访问 /zh/... 展示中文
- [x] 默认语言检测：根据浏览器语言偏好自动选择初始语言
- [x] 翻译文件完整：en.json 和 zh.json 包含所有静态文案 key
- [ ] 所有页面中英文完整：Auth、Entries、Arena、Dashboard 所有页面翻译覆盖 100%
- [ ] 动态内容翻译：用户生成内容（技能描述、评论等）支持多语言
- [ ] RTL 支持（未来）：为阿拉伯语等 RTL 语言预留布局适配能力

## UI 验收

- [x] 语言切换组件位置合理（Header 右上角）
- [x] 语言切换组件样式与整体设计一致
- [ ] 中英文排版无溢出/截断
- [ ] 长文本（如德语等未来语言）布局不崩溃
- [ ] 日期/数字格式按 locale 正确格式化

## 性能验收

- [x] 语言切换响应时间 < 200ms
- [ ] 翻译文件按需加载（不一次性加载所有语言）
- [ ] 翻译 key 缺失时优雅降级（fallback 到默认语言）

## 测试验收

- [ ] 语言切换组件单元测试
- [ ] locale 路由中间件单元测试
- [ ] E2E 中英文切换流程测试
- [ ] 翻译 key 完整性自动化检查（CI 集成）
