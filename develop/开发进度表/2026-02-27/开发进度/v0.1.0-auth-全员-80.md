# Auth 认证系统开发进度

## 模块概述

Supabase Auth 集成，支持 Email + Google + Web3 Wallet 三种认证方式，RBAC 五级角色权限体系（visitor / user / creator / admin / super_admin）。作为平台基础设施层，为所有业务模块提供统一的身份认证与权限控制能力。

## 已完成项

- ✅ Supabase 客户端封装（browser + server）
- ✅ Email + Password 登录/注册
- ✅ Google OAuth 登录
- ✅ JWT 验证中间件
- ✅ 角色权限系统（5 级）
- ✅ 登录/注册页面 UI
