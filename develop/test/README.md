# 测试记录索引 & 测试规范

## 1. 测试规范概述

- **测试框架**：Vitest ^3.1
- **测试类型**：单元测试、集成测试、E2E 测试
- **命名规范**：`*.test.ts` 或 `*.spec.ts`
- **目录结构**：测试文件与源码同目录（collocated）

## 2. 覆盖率要求

- 新代码覆盖率 ≥80%
- 核心模块（core/）≥90%
- 工具函数（utils/）≥95%
- UI 组件 ≥70%

## 3. 测试报告格式规范

- **文件命名**：`v{版本}-{模块}-测试报告.md`
- **存放路径**：`test/{日期}/`
- 必须包含 YAML frontmatter

## 4. 测试命令

```bash
pnpm test              # 运行所有测试
pnpm test:coverage     # 覆盖率报告
pnpm test -- --watch   # 监听模式
```
