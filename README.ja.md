## OpenFinClaw とは？

OpenFinClaw は [OpenClaw](https://github.com/openclaw/openclaw) の**金融ドメイン特化版**です。OpenClaw の実績あるエージェント基盤 — 常時稼働ゲートウェイ、25以上のメッセージングチャネル、プラグインシステム、ベクトルメモリ、マルチモデル対応 — の上に、深い金融インテリジェンスレイヤーを構築します。

**私たちのミッション**: すべての個人に、プロアクティブで、プライベートで、進化し続ける、真にあなただけのAI金融コンパニオンを届けること。

```
従来の金融ツール                 OpenFinClaw
───────────                     ──────────────────────
問い合わせに応答する       →    観察し、学習し、行動する
画一的なアドバイス         →    あなたの人生に合わせた深いパーソナライズ
データはサイロに閉じ込め   →    あなたのデータ、あなたのデバイス、あなたのルール
固定された機能            →    自己進化するスキル
バラバラのツール          →    統一されたライフサイクルコンパニオン
```

## なぜ OpenFinClaw なのか？

2026年、AIエージェントは「呼び出すツール」から「共に生きるコンパニオン」へと変化しています。投資、予算管理、税務計画、取引、退職 — 金融生活は汎用チャットボットや静的ダッシュボードでは対応できないほど複雑でパーソナルです。

OpenFinClaw の違い:

- **24時間365日稼働** — OpenClawのGatewayによりデバイス上で常時稼働。グローバル市場は眠りません。あなたの金融コンパニオンも同様です。
- **自己進化** — エージェントが実行時に新しいスキルを作成しホットロード（JITプラグイン）。市場が変わった？エージェントは数ヶ月ではなく数分で適応します。
- **メモリ駆動** — 4層コンテキストシステム（Soul → Tools → User → Session）により、リスク許容度、過去の意思決定、財務目標をすべての会話を通じて記憶します。
- **プライバシーファースト** — すべてのデータはデバイス上に保持。ベクトルメモリ、取引認証情報、財務履歴 — あなたが選択しない限り外部に出ません。
- **オープンソース** — ベンダーロックインなし。コードを監査し、フォークし、拡張できます。これはあなたの金融主権です。

## 主要機能

### 金融コア機能

| 機能 | 説明 | ステータス |
|------|------|-----------|
| **進化する金融スキル** | 自己改善するスキルプラグイン：ポートフォリオ分析、リスク評価、税務最適化、予算管理など。時間とともに成長し適応します。 | 計画中 |
| **プロアクティブ金融バトラー** | 受動的な応答だけでなく、市場を監視し、定期的な健全性チェックを実施し、予測アラートを送信し、レポートを自動生成します。 | 計画中 |
| **自律取引 (CCXT)** | Hyperliquid、Binance、OKX、Bybit で人間による承認セーフガード付きで取引を実行します。 | 計画中 |
| **ディープ金融エキスパートSDK** | プロフェッショナルグレードの分析、機関投資家レベルのインサイト、定量ツール（SDK キー登録制）。 | 計画中 |
| **スマート情報フローSDK** | キュレーションされたリアルタイム市場インテリジェンス、センチメント分析、決算データ、マクロ指標（SDK キー登録制）。 | 計画中 |
| **ユーザーメモリエンジン** | 財務プロファイル、リスク選好、投資傾向、意思決定履歴を記憶。対話のたびにスマートになります。 | 計画中 |

### アドバンスドインテリジェンス

| 機能 | 説明 | ステータス |
|------|------|-----------|
| **セルフレビューエンジン** | 取引/意思決定後に、結果と予測を自動レビューし、認知バイアスを特定し、自身の意思決定パラメータを更新します。 | 計画中 |
| **予測アラート** | 価格アラートを超えて、クジラの動き検知、過去パターンとの相関、支出行動に基づく予算超過予測を行います。 | 計画中 |
| **行動ファイナンスガード** | 感情的な取引パターン（リベンジトレード、FOMO、パニック売り）を検知し、冷静でエビデンスに基づいたガイダンスを提供します。 | 計画中 |
| **金融デジタルツイン** | あなたの完全な財務未来をシミュレーション。「この家を買ったら？」「いつ引退できる？」「この投資は10年後の見通しをどう変える？」 | 計画中 |
| **マルチエージェント金融チーム** | リサーチャー、トレーダー、リスクオフィサー、コンプライアンス監査員。各々が独自の人格、権限、メモリを持ち、あなた専属の金融チームとして協働します。 | 計画中 |
| **規制コンプライアンス自動化** | 税務イベントの自動追跡、大口ポジション報告、クロスボーダーコンプライアンスチェック（CRS/FATCA）、監査証跡の生成。 | 計画中 |

### OpenClaw の基盤

上記すべての機能は OpenClaw の実績ある基盤の上に構築されています:

- **25以上のメッセージングチャネル** — WhatsApp、Telegram、Slack、Discord、Signal、iMessage、Teams など
- **常時稼働ゲートウェイ** — 24時間365日デーモンプロセス、ホットリロード対応、再起動不要
- **Piプリミティブエンジン** — 4つのコアプリミティブ (Read/Write/Edit/Bash) + LLM = 無限の組み合わせ
- **JITプラグインシステム** — エージェントが実行時にTypeScriptプラグインを作成しホットロード可能
- **ベクトルメモリ** — sqlite-vecベースのハイブリッド検索（セマンティック + キーワード）で長期記憶を実現
- **マルチモデル対応** — Claude、GPT、Gemini、Ollama など10以上のプロバイダー、自動フェイルオーバー付き
- **5層セキュリティ** — ネットワーク → 認証 → チャネル → 実行 → デバイス、ヒューマンインザループ承認付き
- **ブラウザ自動化** — Playwrightベース、ヘッドレス + 拡張リレーモード、APIのないサイトにも対応

## 設計原則

1. **ユーザーファースト** — すべての機能はユーザーの財務的幸福のために。エンゲージメント指標のためではありません。
2. **プライバシーバイデザイン** — 金融データはデバイス上に保持。例外なし。
3. **オープンソース** — コアエンジンはオープン。プレミアムデータ/エキスパートアクセスはSDKキーで提供。
4. **安全ガードレール** — 取引操作は常にユーザーの明示的な承認が必要。サイレント取引は行いません。
5. **進化し続ける** — スキルと機能はコミュニティの貢献と自己改善を通じて成長します。
6. **透明性** — 結果だけでなく、推論プロセスを提示。ユーザーは推奨の理由を理解すべきです。
